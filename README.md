# ⚛️ React Quiz App

Інтерактивний додаток-вікторина, створений для перевірки знань за допомогою тестових запитань. Проєкт розроблено під час вивчення основ бібліотеки React.

🔗 **[Дивитись готовий проєкт (Live Demo)](https://fediaiandrii.github.io/react-quiz/)** <!-- Заміниш # на посилання від GitHub Pages -->

## 🚀 Функціонал

- Завантаження пулу питань із локального JSON-файлу.
- Інтерактивний таймер на проходження тесту.
- Підрахунок балів та система максимального рекорду (Highscore).
- Візуалізація прогресу проходження вікторини.
- Різні стани додатку (завантаження, помилка, старт, активний процес, фініш).

## 🛠 Технології

- **React** (Функціональні компоненти, Hooks: `useState`, `useEffect`, `useReducer`)
- **JavaScript (ES6+)**
- **HTML5 / CSS3**

## 💻 Як запустити локально

Якщо ви хочете завантажити проєкт і запустити його на своєму комп'ютері, виконайте наступні кроки:

1. Клонуйте репозиторій:

   ```bash
   git clone https://github.com/ТВІЙ_НІК/react-quiz.git
   ```

2. Перейдіть у папку з проєктом:

   ```bash
   cd react-quiz
   ```

3. Встановіть необхідні залежності:

   ```bash
   npm install
   ```

4. Запустіть локальний сервер:
   ```bash
   npm start
   ```

Додаток буде доступний за адресою `http://localhost:3000` у вашому браузері.
